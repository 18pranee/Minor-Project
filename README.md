# 🌱 Plant Disease Detection System

## 📌 Description
Developed an Explainable Deep Learning framework for plant disease detection using EfficientNet and Vision Transformer ensemble.

## 🚀 Features
- Detects plant diseases from leaf images
- Uses EfficientNet + Vision Transformer
- Explainable AI visualization

## 🛠 Tech Stack
- Python
- TensorFlow / PyTorch
- OpenCV

## 📂 Project Structure
- app.py
- train_model.py
- predict.py

## ▶️ How to Run
1. Install dependencies:
   pip install -r requirements.txt
2. Run:
   python app.py
